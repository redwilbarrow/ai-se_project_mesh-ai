# MeshAI

MeshAI is a web platform that helps users get answers from their documents using AI. Users can upload PDF files to build their own knowledge base, then ask questions in a chat interface and receive answers based on the uploaded content.

## What MeshAI does

- Upload PDF documents
- Build a personal knowledge base from uploaded files
- Ask questions in a chat experience
- Get AI-generated answers grounded in the uploaded documents

## Project goals

MeshAI is designed to make document-based information retrieval simple and intuitive. Instead of manually searching through long files, users can ask natural-language questions and quickly find relevant answers.

## Tech stack

- Node.js
- Express
- TypeScript
- ESLint and Prettier

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the health check endpoint in your browser:
   ```bash
   [npm run dev](http://localhost:3000/health)
   ```

## Current status

This project currently includes a TypeScript Express backend scaffold with routes and stubbed responses for authentication, chats, documents, and query endpoints.
